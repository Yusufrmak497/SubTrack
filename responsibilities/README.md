# Responsibilities Folder Guide

Create one file per team member using their exact student number:

- `responsibilities/23023023.md`
- `responsibilities/23024567.md`
- ...

Each file should include:
1. Role in project
2. Assigned tasks
3. Delivered outputs (with file references)
4. Technical decisions taken
5. Evidence of completion (commits, tests, demo scope)

A starter template is included in `responsibilities/23023023.md`.
If your real student number is different, duplicate/rename that file.
