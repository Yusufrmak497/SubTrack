Create a clean, modern, single-page website called **TinyVault** using only HTML, CSS, and vanilla JavaScript (no frameworks).

Requirements:
1. Build semantic page structure with header, main sections, and footer.
2. Show at least 6 subscription cards (Netflix, Spotify, Notion, Google Drive, YouTube Premium, Duolingo).
3. Each card must include:
   - service name
   - category (Entertainment, Music, Productivity, Cloud, Education)
   - billing cycle (Monthly/Yearly)
   - price
   - next payment date
4. Add a summary area that displays:
   - active subscription count
   - estimated monthly total (convert yearly to monthly estimate)
5. Add polished styling:
   - responsive layout
   - card hover effects
   - clear color coding for categories
6. Add a dark mode toggle button with JS.

Keep code beginner-friendly and easy to explain in class.
